# 工具发布平台 - 视觉设计规格

## 设计概念

**"精密工坊"美学** - 现代化的工具陈列空间，灵感来自精密仪器面板和高端电子工作台。

### 核心特征
- 深色主题：营造专注、专业的工具使用氛围
- 代码美学：微妙的等宽字体元素，呼应开发者工具的本质
- 精密感：细致的边框、精确的对齐、几何图形装饰
- 呼吸感：大量留白，让工具本身成为焦点

## 色彩系统

### 主色调
```css
--bg-primary: #0d0d0d;        /* 深邃黑 - 主背景 */
--bg-secondary: #141414;      /* 工具卡片背景 */
--bg-tertiary: #1a1a1a;       /* 悬浮状态 */
--border-subtle: #2a2a2a;     /* 细边框 */
--border-highlight: #3a3a3a;  /* 高亮边框 */
```

### 强调色
```css
--accent-primary: #00d4aa;    /* 青绿色 - 主要交互 */
--accent-secondary: #f59e0b;  /* 琥珀色 - 精选/特殊标记 */
--accent-glow: rgba(0, 212, 170, 0.15); /* 发光效果 */
```

### 文字色
```css
--text-primary: #f5f5f5;      /* 主要文字 */
--text-secondary: #a3a3a3;    /* 次要/描述文字 */
--text-muted: #737373;        /* 微弱文字 */
```

## 字体系统

### 字体选择
- **标题字体**: `Space Grotesk` - 几何感、现代、技术感
- **正文字体**: `JetBrains Mono` - 等宽、清晰、开发者友好
- **备用**: `system-ui, sans-serif`

### 字体比例
```
英雄区标题: 4rem / 600 / 1.1 / -0.02em
页面标题:   2.5rem / 600 / 1.2 / -0.01em
工具名称:   1.125rem / 600 / 1.3
描述文字:   0.875rem / 400 / 1.6
标签文字:   0.75rem / 500 / 1 / 0.05em
```

## 组件设计

### 1. 导航栏
- 固定顶部，高度 64px
- 背景: `bg-primary` + 底部边框
- 左侧: Logo + 平台名称
- 右侧: GitHub 链接（图标）
- 滚动时添加微妙阴影

### 2. 英雄区
- 大标题 + 副标题描述
- 居中布局
- 装饰性几何图形（角落的细线条）
- 底部搜索栏突出显示

### 3. 搜索栏
- 大号输入框，居中放置
- 深色背景 + 青绿发光边框（focus状态）
- 左侧搜索图标
- 实时过滤反馈

### 4. 分类标签
- 胶囊形状（pill）
- 默认: 透明背景 + 边框
- 选中: 青绿背景 + 深色文字
- 悬浮: 边框变亮

### 5. 工具卡片
```
┌─────────────────────────────────┐
│ [ICON]                          │ ← 工具图标（emoji或SVG）
│ 工具名称                        │ ← 粗体，主要文字色
│ 简短的工具描述...               │ ← 次要文字色，2行截断
│                                 │
│ [开发者工具] [在线]             │ ← 分类标签
└─────────────────────────────────┘
```

**卡片细节:**
- 背景: `bg-secondary`
- 边框: 1px solid `border-subtle`
- 圆角: 12px
- 悬浮: 边框变亮 + 轻微上移 + 阴影
- 点击: 新窗口打开工具

**精选卡片:**
- 边框: `accent-primary`
- 左上角 "精选" 标记

### 6. 网格布局
- 移动端: 1列
- 平板: 2列
- 桌面: 3列
- 大屏: 4列

## 动效规格

### 入场动画
- 页面加载: 内容淡入 + 轻微上移
- 卡片: 错开入场，每张延迟 50ms
- 时长: 600ms
- 缓动: `cubic-bezier(0.16, 1, 0.3, 1)` (ease-out-expo)

### 悬浮效果
- 卡片上移: -4px
- 边框颜色过渡: 200ms ease
- 阴影显现: `0 8px 32px rgba(0, 212, 170, 0.1)`

### 搜索交互
- 输入框focus: 青绿边框发光动画
- 结果过滤: 卡片淡出/淡入，300ms
